# MATLAB 辅助脚本

## THD_Calc.m

读取双通道示波器 CSV 波形文件（`WAVE.csv`，从第 20 行开始），计算各通道的 **RMS** 与 **THD**（dB / %），并绘制时域波形和单边频谱图（上限 200 kHz）。

**用途**：快速评估交流侧电压/电流的谐波质量。

## LinearCompensation.m

线性校准补偿脚本。输入 ADC 转换的原始斜率/偏移，配合测量值与实际值数据对，通过线性回归计算更新后的斜率和偏移，并绘制校准曲线（数据点、实际值线、原始线、补偿线）。

**用途**：标定 ADC 采样通道的增益和偏置误差（支持电压/电流通道）。
